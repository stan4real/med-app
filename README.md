# Проект на React с TypeScript, React Router, Vite и Styled-Components

## Описание проекта

Это веб-приложение, реализованное с использованием **React**, **TypeScript**, **React Router**, **Vite** и **Styled-Components**. В проекте реализована маршрутизация с тремя основными страницами: **Главная**, **Контакты** и **Страница пользователя**. Страница пользователя защищена и доступна только для авторизованных пользователей.
Верстка Pixel Perfect выполнена по макету с figma. Данные о пользователе хранятся в **localStorage**, поэтому при удалении данных о пользователя старница будет перезагружена.

## Технологии

- **React** — библиотека для построения пользовательских интерфейсов.
- **TypeScript** — статическая типизация для JavaScript.
- **React Router** — библиотека для маршрутизации в React.
- **Vite** — сборщик и инструмент для разработки с высокой производительностью.
- **Styled-Components** — библиотека для стилизации компонентов с использованием CSS в JS.

## Структура проекта

- **/src** — исходный код приложения
  - **/components** — компоненты приложения
    - **/pages** — страницы приложения
    - **/ui** — элементы интерфейса
  - **/data** — json с пользователями для авторизации и статичные данные
  - **main.tsx** — точка входа в приложение; основной компонент приложения, содержащий маршрутизацию
- **/public** — статичные файлы (например, `index.html`, изображения)
